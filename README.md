# UMSI Career Development Resource Hub

Updated project files for the SI 539 client project.

## Included deliverables
- Home page
- Resume page
- Networking and Building Connections page
- Portfolio page (chosen topic page)
- 404 page
- Shared mobile-first responsive stylesheet

## Notes
- Images are stored in `images`.
- Shared styles are in `assets/styles.css`.
- The HTML includes a skip link, semantic landmarks, and consistent navigation/footer.

## Local setup

1. Clone the repository
2. Open the project folder in VS Code
3. Launch `index.html` with Live Server or open it directly in your browser

## Site map

- `index.html` - homepage
- `resume.html` - resume support resources
- `networking.html` - networking guidance and outreach help
- `portfolio.html` - portfolio planning help
- `404.html` - custom error page
- `assets/styles.css` - shared site styles

## Goals for this resource hub

This site helps UMSI students quickly find practical career development support in one place. The project focuses on:

- clear navigation
- mobile-friendly layouts
- accessible page structure
- concise, actionable content

What makes this site useful?

  • Clear starting points: topic cards and quick links help you decide what to do first.
  • Context before links: each resource includes a short explanation so you know why it matters.
  • Student-centered guidance: content is written for UMSI students preparing for internships and full-time roles.
  • Accessible design: the site uses semantic structure, visible focus states, alt text, and responsive layouts.

Frequently asked question

Where should I start if I am overwhelmed?

Begin with one goal for this week: improve your resume, send one networking message, or select two portfolio pieces. Then use the linked CDO resources on that page and book a coaching appointment if you want feedback.
